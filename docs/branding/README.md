# Branding — MyClub

## Objetivo

Esta pasta consolida a definição de marca, identidade visual e design system do MyClub.

O material aqui documentado serve como base para:

- decisões de produto e comunicação
- criação de logo e aplicações visuais
- implementação de interface administrativa
- alinhamento futuro com apps web públicos, mobile e materiais comerciais

## Direção geral da marca

O MyClub não deve parecer:

- improvisado
- excessivamente corporativo
- genérico como um SaaS financeiro
- infantilizado
- “esportivo” apenas por usar verde neon, escudos aleatórios e bolas como ornamento

O MyClub deve parecer:

- confiável
- operacional
- competitivo
- organizado
- contemporâneo
- próximo da cultura real do futebol amador e semiprofissional

---

## Princípio de consistência

Toda decisão visual futura deve conseguir responder a estas três perguntas:

1. Isso reforça organização, disputa e pertencimento?
2. Isso serve tanto para produto operacional quanto para presença pública?
3. Isso consegue escalar para múltiplas superfícies sem perder reconhecimento?

Se a resposta for “não” para qualquer item, a decisão deve ser revisada.
